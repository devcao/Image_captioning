# Image Captioning
A demo of Google's 2014 Image Captioning paper.
To run, 
```
git clone git@github.com:statcao/Image_captioning.git
cd Image_captioning
```
Then, you can either run the Jupyter Notebook interactively, or 

```
python train.py
```
After training is done, 

```
python visualize.py
```
